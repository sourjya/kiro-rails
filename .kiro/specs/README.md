# Kiro Specs Index

Chronological index of all feature specifications. Each spec contains
`requirements.md`, `design.md`, and `tasks.md`.

---

## Spec Status Legend

- **Complete** - All tasks done, merged to main
- **In Progress** - Tasks being executed on a feature branch
- **Planned** - Requirements and design written, not yet started
- **Superseded** - Retired; work absorbed into another spec
